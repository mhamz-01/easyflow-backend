# Private and Public projet

1. While creating a project give an option to the creater if the project is private or public, private projects will only be visible to people who are assigned/invited to that project
2. While the public projects will be visible to all the members with their respective permission access

# Access Permission

3. Limited Member can only access items shared with them
4. Can access items in the projects
5. Manager can create,update,view items
6. Can go to settings, delete items and do the rest
